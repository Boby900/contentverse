<h2>🛠️ Installation Steps:</h2>

<p>1. clone</p>

```
git clone https://github.com/Boby900/clientverse  && cd clientverse
```

<p>2. add the necessary envs</p>

```
.env .env.local
```

<p>3. Install dependencies</p>

```
npm i
```

<p>4. start local dev server</p>

```
npm run dev
```
